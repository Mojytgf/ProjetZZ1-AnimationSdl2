# Animation C/SDL2

Projet de mini-jeux en C/SDL réalisé dans le cadre de la première année à l'ISIMA.  
Ce dépôt regroupe plusieurs projets réalisés individuellement par les étudiants : animations de sprites, jeux d’arcade, jeu du serpent, voyageur de commerce, et interface graphique avec SDL/X11.

## Structure du projet

- `animer_sprite/` : animations de sprites  
  - `animer_sprite_Abdelkader/`  
  - `animer_sprite_aya/`  
  - `animer_sprite_mohammed/`  

- `jeu_d_arcade/` : mini-jeux d’arcade  

- `pave_serpent/` : projet du serpent  

- `voyageur_de_commerce/` : algorithmes pour le problème du voyageur de commerce (C et recuit simulé)  

- `X_fenetre/` : projets utilisant SDL et X11 pour interface graphique  

## Installation

1. Cloner le dépôt :  
   ```bash
   git clone <URL_DU_REPO>
Compiler les projets individuellement avec gcc ou via make si disponible.
Exemple pour voyageur_de_commerce :

```bash
   cd voyageur_de_commerce
   make
   ./mon_programme
```

⚠️ Les projets contiennent certains fichiers vidéos et binaires pour démonstration. Seuls les fichiers source .c et .h sont nécessaires pour la compilation.
